# Typography

The Cooder Typography is designed to provide enough flexibility for your application while also remaining easy to use. You should use only these styles in your application unless you have a strong need for additional styles. The Cooder theme adds several additional variants when compared to the defauly MUI theme.

View [Mui Typography](https://mui.com/system/typography/#variant) for more details.

### Usage

```tsx
<Typography variant="h1">h1</Typography>
<Typography variant="caption1">caption1</Typography>
<Typography variant="body2">body2</Typography>
...
```
