# Shadows

The helpers allow you to control relative depth, or distance, between two surfaces along the z-axis. By default, there are 25 elevation levels.
View [MUI Shadows](https://mui.com/system/shadows/) for more details.

### Usage

```tsx
<Box sx={{ boxShadow: 0 }} /> // box-shadow: none
<Box sx={{ boxShadow: 1 }} /> // box-shadow: 0px 8px 24px -16px rgba(0, 0, 0, 0.25)
<Box sx={{ boxShadow: 2 }} /> // box-shadow: 0px 2px 1px -1px rgba(0,0,0,0.12),0px 1px 1px 0px rgba(0,0,0,0.06),0px 1px 3px 0px rgba(0,0,0,0.08)
<Box sx={{ boxShadow: 3 }} /> // box-shadow: 0px 3px 1px -2px rgba(0,0,0,0.12),0px 2px 2px 0px rgba(0,0,0,0.06),0px 1px 5px 0px rgba(0,0,0,0.08)
...
```
